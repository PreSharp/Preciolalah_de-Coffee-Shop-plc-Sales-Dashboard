#  Coffee Shop Sales Dashboard
##  Project Overview

This project presents an interactive **Coffee Shop Sales Dashboard** developed in Microsoft Excel to analyze sales performance, customer purchasing behavior, product trends, and business insights. The dashboard transforms raw transactional data into meaningful visualizations that support data-driven decision-making.

##  Project Objectives

* Analyze overall sales performance.
* Identify top-selling products and categories.
* Track monthly and daily sales trends.
* Monitor customer purchasing patterns.
* Provide actionable insights for business growth.
* Create an interactive and user-friendly dashboard.

##  Dataset Information

The dataset contains coffee shop transaction records including:

* Transaction ID
* Date and Time
* Product Name
* Product Category
* Quantity Sold
* Unit Price
* Total Sales
* Store Location

##  Data Cleaning Process

The following data cleaning steps were performed in Microsoft Excel before building the dashboard:

### 1. Removed Duplicate Records

* Checked for duplicate transactions.
* Used Excel's **Remove Duplicates** feature to ensure data integrity.

### 2. Handled Missing Values

* Identified blank cells.
* Filled missing information where applicable.
* Removed incomplete records that could impact analysis.

### 3. Standardized Data Formats

* Converted dates into a consistent format.
* Standardized text entries for products and categories.
* Corrected inconsistent naming conventions.

### 4. Validated Numerical Data

* Checked quantities and sales values for accuracy.
* Corrected data entry errors where necessary.

### 5. Created Additional Fields

* Extracted:

  * Month
  * Day of Week
  * Hour of Transaction
* Generated calculated fields for enhanced analysis.
